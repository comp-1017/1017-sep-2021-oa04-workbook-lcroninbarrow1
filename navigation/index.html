Create navigational links within a website

There are several techniqies to navigating within a website. the semantics tag for navigation is the <nav> tag. you can navigate through your website by using relative addressing for our examples we will use the unordered list tag <ul> tag. our example will consist of:

  • page to page within the same folder
  • page to page between folders
  • staying on the current page
  • going to a specific place on the current page(on page naviagation)
  • going to a specific place on another page
